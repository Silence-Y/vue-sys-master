# vue-sys-master
基于Vue+Element-ui开发的后台管理系统模板

## Build Setup
1. 安装依赖
``` bash
npm install
```
2. 本地运行
``` bash
npm run dev
```
3. 编译打包
``` bash
npm run build
```
